Prima di eseguire qualsiasi script, eseguire i seguenti comandi da terminale:

Windows:
git clone https://github.com/simuan02/guitar_hero_rl.git
cd guitar_hero 
.\scripts\restore.ps1

Mac/Linux:
git clone https://github.com/simuan02/guitar_hero_rl.git
cd guitar_hero
chmod +x scripts/restore.sh
./scripts/restore.sh
